# 🎮 Doraemon Interaction Guide

## 3 Ways to Interact

### 1. 🎤 Voice Input
**How:** Click mic button → Speak → Agent responds with voice

**Best for:**
- Hands-free operation
- Natural conversation
- Quick commands
- Multitasking

**Example:**
```
Click 🎤 → Say "Add task buy groceries" → Agent confirms
```

---

### 2. ✍️ Text Input
**How:** Type in text box → Press Enter or click ➤ → Agent responds with voice

**Best for:**
- Silent environments
- Precise input
- Complex messages
- When mic unavailable

**Example:**
```
Type "What's on my schedule?" → Press Enter → Agent speaks response
```

---

### 3. 🖱️ Manual Controls
**How:** Click buttons in sidebar → Instant action

**Best for:**
- Quick task management
- Editing mistakes
- Batch operations
- Visual task review

**Example:**
```
Click + → Type "Buy milk" → Click "Add Task"
```

## 🎯 Best Practices

### Voice
- ✅ Speak clearly
- ✅ Pause between commands
- ✅ Use natural language
- ❌ Don't shout
- ❌ Don't speak too fast

### Text
- ✅ Type complete sentences
- ✅ Use natural language
- ✅ Press Enter to send
- ❌ Don't use abbreviations
- ❌ Don't type while voice active

### Manual
- ✅ Hover to see actions
- ✅ Click once
- ✅ Wait for confirmation
- ❌ Don't double-click
- ❌ Don't spam buttons
