# 🚀 Quick Start Guide - Doraemon AI Voice Agent

## ⚡ 5-Minute Setup

### Step 1: Start Backend (Terminal 1)
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
✅ Backend running at: http://localhost:8000

### Step 2: Start Frontend (Terminal 2)
```bash
cd frontend
npm install
npm run dev
```
✅ Frontend running at: http://localhost:5173

### Step 3: Open Browser
Go to: **http://localhost:5173**

---

## 🎤 How to Use

### 1️⃣ Click the Blue Orb or Mic Button
The agent will greet you!

### 2️⃣ Start Talking
Try these commands:

**Add Tasks:**
- "Add task buy groceries"
- "Add task call mom"
- "Remind me to exercise"

**View Tasks:**
- "Show my tasks"
- "List my tasks"
- "What are my tasks"

**Delete Tasks:**
- "Delete task buy groceries"
- "Complete task call mom"

**Save Memories:**
- "Remember my exam is Monday"
- "Remember I like pizza"
- "Don't forget my birthday is May 15"

**Recall Memories:**
- "What did I tell you?"
- "What do you remember about me?"

**End Conversation:**
- "Goodbye"
- "Thank you"
- "Bye"

### 3️⃣ Watch the Magic! ✨
- See tasks appear in the sidebar
- Memories are saved automatically
- Full conversation log is displayed

## 🐛 Quick Fixes

**Mic not working?**
- Check browser permissions
- Refresh the page
- Try Chrome/Edge

**Backend error?**
- Make sure port 8000 is free
- Check if Python is installed
- Verify you're in the backend folder

**Frontend error?**
- Make sure port 5173 is free
- Check if Node.js is installed
- Run `npm install` again
